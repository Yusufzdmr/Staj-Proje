# 🪪 Ziyaretçi Kayıt Sistemi — Staj Projesi

Staj döneminde **Microsoft Power Platform** (Power Apps + Power Automate) ile
geliştirilen bir **Ziyaretçi Kayıt Sistemi** uygulamasıdır.

![Power Apps](https://img.shields.io/badge/Microsoft-Power%20Apps-742774?logo=powerapps&logoColor=white)
![Power Automate](https://img.shields.io/badge/Microsoft-Power%20Automate-0066FF?logo=powerautomate&logoColor=white)

## 📖 Genel Bakış

Kurum girişlerinde ziyaretçi bilgilerinin dijital ortamda kaydedilmesini sağlayan,
düşük kodlu (low-code) bir uygulama. Power Apps ile oluşturulan arayüz ve
Power Automate akışı (flow) içerir.

## 📦 İçerik

```
ZiyaretçiKayıtSistemi_*.zip   → Power Platform dışa aktarım paketi
  ├── Microsoft.PowerApps/    → Canvas uygulama (.msapp)
  └── Microsoft.Flow/         → Power Automate akış tanımı
```

## 🚀 İçe Aktarma

1. [make.powerapps.com](https://make.powerapps.com) → **Apps → Import canvas app**
2. `.zip` paketini yükle
3. Bağlantıları (connections) eşle → **Import**

> Bu paket, Power Platform'dan dışa aktarılmış uygulamanın taşınabilir halidir;
> bir Microsoft 365 / Power Platform ortamı gerektirir.
